# Shipyard

Controlled build room for Codi/Codex work.

Shipyard turns rough ideas into tracked build cards, moves work through a visible floor, requires Gate approval before shipping, and saves approved decisions into Hangar memory.

## Current Build

MGP v0.1 local-first prototype.

## Sections

- Inlet: capture rough ideas and turn them into task cards
- Floor: queue and track working cards
- Gate: approve, tweak, or scrap work
- Hangar: save approved decisions and reusable memory

## Storage

This prototype stores data in browser localStorage and can export JSON.
